You can find code example in `test/test.txt`

```bash
$ cat test/test.txt > input.txt
```

Or you can link the test file

Linux:
```bash
$ ln -s test/test.txt input.txt
```

Windows:
```cmd
$ mklink input.txt test/test.txt
```
